Company Directory

This project involves creating a company directory web application with full CRUD (Create, Read, Update, Delete) functionality for managing personnel. The application will feature a user-friendly interface optimized for both mobile and desktop platforms, enabling efficient search and filtering of staff, departments, and locations. It will utilize PHP and MySQL or MariaDB for backend operations, with a front-end built using HTML, CSS (preferably with a framework), JavaScript, and jQuery. The design must adhere to professional standards, accommodating dynamic employee data and departmental changes. The final product will demonstrate your ability to deliver a robust, scalable solution that integrates seamlessly with the provided database schema.
