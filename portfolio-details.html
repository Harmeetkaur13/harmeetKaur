Gazetteer

The Gazetteer project is a mobile-first web application designed to offer detailed profiling of countries by leveraging multiple third-party APIs. The application delivers demographic, climatic, geographical, and other critical data in a user-friendly interface, adaptable for both mobile and desktop use. By integrating services like Geonames, OpenCage, Open Weather, and Rest Countries, the Gazetteer provides users with essential information about any country, including its capital city, population, currency, current weather, and more. The project emphasizes the importance of responsive design, utilizing frameworks like Bootstrap and libraries like Leaflet to ensure a seamless user experience across all devices. This project serves as a portfolio piece to showcase web development skills and the ability to integrate complex data sources into a cohesive application.
